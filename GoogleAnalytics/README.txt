Installation
============

Manual Install

1) Extract the contents of this zip file on your computer
2) Upload the zip's content to cc-content/plugin
3) Enable the plugin (Admin Panel -> Plugins)
4) Enter your Google Analytics Tracking Id (Admin Panel -> Plugins -> Google Analytics -> Settings)

Automatic Install

1) Upload the zip file via the Admin Panel (Admin Panel -> Plugins -> Add New Plugin)
2) Enter your Google Analytics Tracking Id (Admin Panel -> Plugins -> Google Analytics -> Settings)
